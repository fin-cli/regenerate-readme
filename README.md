fp-cli/regenerate-readme
========================

PHP Composter action to regenerate the README.md file (precommit).



Quick links: [Installing](#installing) | [Using](#using) | [Contributing](#contributing) | [Support](#support)

## Installing

Just add as a development requirement to your `composer.json`, and it should work automagically:

```BASH
composer require --dev fp-cli/regenerate-readme
```

## Using

It should just work when you `git commit`.
